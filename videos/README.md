# Dossier vidéos

Déposez simplement vos vidéos MP4 ici en suivant ce nommage :

- `video1.mp4`
- `video2.mp4`
- `video3.mp4`
- ...

Le site les détecte automatiquement et les affiche dans la section Portfolio.

## Conseils

- Format recommandé : `1080x1920` (9:16)
- Codec recommandé : H.264
- Taille conseillée : < 100 MB par vidéo (limite GitHub)
- Évitez les espaces et caractères spéciaux dans les noms de fichiers
