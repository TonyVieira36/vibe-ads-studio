# Vibe Ads Studio

Site statique GitHub Pages optimisé conversion (copywriting + FAQ + process + conformité légale) avec portfolio MP4 auto-détecté.

## Portfolio automatique

Le site détecte automatiquement les vidéos dans `videos/` selon le pattern :

- `video1.mp4`
- `video2.mp4`
- `video3.mp4`
- etc.

Aucune modification de code n'est nécessaire tant que vous respectez ce nommage.

## Structure

```text
vibe-ads-studio/
├── index.html
├── mentions-legales.html
├── cgv.html
├── politique-confidentialite.html
└── videos/
    ├── README.md
    └── .gitkeep
```
